Basics

Creating the CSS File

To create the CSS file, create a file ending in .css and place it inside a folder named css.

Attaching the CSS File

To attach the .css file to the HTML, the line written below must be placed within the head.

<link rel="stylesheet" href="css/style.css">

The other option is to simply write link and the line should autofill.

CSS Syntax

The entire block is called a rule set. A selector defined which HTML element is being targeted. A declaration consists of a property and a value, and ends with a semicolon. A property is how the elemet will be altered, while a value is what the element will be altered to.

h1 {

colour: black;

}

Font Properties

Font Family

Font Size

Changes the size of the font. Value is a single number and can be in rem, em or px, however rem is preffered.

font-size: 3rem;

Font Weight

Changes the weight of the text.

font-weight: bold;

Font Style

Changes the style of the text.

font-style: italic;

Font Height

Changes the height of the text. Value is a multiplier.

line-height: 1.5;
